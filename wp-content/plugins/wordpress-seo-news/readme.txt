== Description ==

This plugin adds Google News XML Sitemaps capabilities to WordPress SEO by Yoast.

== Installation ==

1. Upload the `plugin` folder to the `/wp-content/plugins/` directory or installed it through the backend.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go into the SEO admin section and enable the XML News Sitemap feature on the SEO -> News SEO page.
